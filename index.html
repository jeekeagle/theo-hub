// 六大子系统
NOTE
笔记系统 theo-note/
所有原始内容的沉淀地。思想的入口，也是 Night Routine 的原材料。
  • note-flow/ — 思想流动地
  • → seed.md 高价值沉淀 核心
  • → future.md 想法种子库 胚芽/发芽中/已行动
  • → qa.md 问题收集箱
  • → Writing-Thoughts.md 写作主题
  • note-auto/ — Night Routine 输出
  • note-topic/ — 主题分类笔记
  • note-wiki/ — LLM Wiki
LAB
探索系统 theo-lab/
新项目孵化地，实验性想法的验证场所。低成本进入，高成本离开。
  • 快速试错，不做深度整理
  • 跑通 → 迁移到输出系统
  • 跑死 → 删掉，不留痕迹
  • 005-ai-kid — AI 幼儿融合教育
  • 饮食脚本 — 迁移至 theo-auto/
SEED
输出系统 theo-output/
最终发布的内容产品。验证通过的想法，在这里变成公开的价值。
  • theo-openwalk/ — 技术博客
  • → AstroPaper 主题
  • → GitHub Pages 托管
  • → Night Routine 自动发布
  • 来自 note-topic 成熟主题
  • 来自 lab 验证通过的实验
AUTO
执行系统 theo-auto/
所有自动化脚本、定时任务、提醒规则。系统的引擎，人不需要介入。
  • Night Routine — 每晚 01:30
  • → 整理 note-flow 当天内容
  • → 让胚芽想法往前走一步
  • → 生成写作主题
  • → 推送发布到 Seed
  • 饮食脚本 — 定时提醒
  • 飞书 CLI — 文档自动同步
TOOL
工具生态 theo-agent/
Agent 的技能库。经验的结晶，遇到特定任务自动加载对应 Skill。
  • 写作技能 — 写作引擎 · 润色 · 翻译
  • 发布技能 — Seed · 微信公众号 · OpenWalk
  • 研究技能 — 论文 · RSS · 圆桌讨论
  • 系统技能 — Night Routine · Skill 管理
  • 所有 Skill 存于 ~/.hermes/skills/
  • 按场景调用，不由人主动选择
FILE
附件库 theo-resource/
所有静态资源文件。不做深度整理，只管存取。
  • theo-picture/ — 图片资源
  • 笔记中引用的图片
  • 信息卡配图
  • Seed 博客文章配图
  • 不做分类，只按时间归档
